<h1> GMS2 Project Line counter</h2>
<br>
<p>With this executable you can find how many ACTUAL lines of code in your GMS2 Project. </p>
<br>
<h2>How It Works.</h3>
<p>This program scans the project directory provided to count the total lines of GML/Shader code <i>*Excluding Comments & Empty Lines*</i>, then gives a total of how many lines the project consists of per category.</p>
<p>There are 4 categories for the scan.</p>
<ul>
  <li>GML Objects</li>
  <li>Shader Files</li>
  <li>Script Assets</li>
  <li>All - (Which includes Room Creation code)</li>
</ul>
<p> There is also a 3rd set of options that let you choose whether or not you want to export the report where you select a location to save it to.</p>
<br>

<h2> How to Use</h3>
<p>Before the scan even starts, there are some options you can choose from depending on what you want the result total to include. 
  You supply the file/file-path or drag in a file.
  Once that is done, it will show you the file path you have selected and ask you what type of counting mode you want to preform.
  You can select 1 of 4 modes for the scan, each will affect the result total.
  Then you can choose whether you want to export the report.

<br>
<h2> Detailed Step-by-Step</h3>
<h3>1. Run the Exe and you will be greeted with this.</h3>
<img width="476" height="176" alt="image" src="https://github.com/user-attachments/assets/11ef4405-a546-48d0-8f7f-5abd7e92a285" />
<p>
You can drag a project folder, use the directory the exe is located in, or paste the file-path where your project is via these options:
</p>
<ul>
  <li>Drag folder onto window</li>
  <li>Use current directory</li>
  <li>Paste full project path</li>
</ul>
<p>Once you have typed in the number of the option you choose, press enter. (In the case you paste a file-path, it will ask you on the next prompt, then press enter again)</p>
<i>Note: If you do put in the file-path, make sure it is the location of where the <code> .yyp </code> is located.</i>
</p>
<h3>2. Once the directory has been selected, it will show the path you have selected.</h3>
<img width="657" height="178" alt="image" src="https://github.com/user-attachments/assets/3f9e0b84-ad73-4474-87e0-a819e82b3078" />
<ul>
  <li>Count only Objects GML</li>
  <li>Count only Shader Files</li>
  <li>Count only Script GML</li>
  <li>Count ALL <i>*which includes room creation code</i></li>  
</ul>
<p>Once you have typed in the number of the option you choose, press enter. <i>*If you chose the wrong option you can just close and re-open the program.*</i></p>
<h3>3. Choose whether or not you want to save the report after the count has finished.</h3>
<img width="213" height="85" alt="image" src="https://github.com/user-attachments/assets/dda17465-890e-4b84-9968-782a878a50f8" />
<p>Once you have typed in the number of the option you choose, press enter.</p>
<h3>4. Project Scan.</h3>
<p>The program will now scan your entire project counting all the lines of code <i>Without comments, or empty lines.</i></p>
<p>This may take a short or long time depending on the size of your project. In most cases it should be done within a minute.</p>
